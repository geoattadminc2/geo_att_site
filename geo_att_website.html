üìÑ Introduction

About the App: GEO_ATT is a smart attendance application designed to record user presence based on location (geofencing) and live photo verification. It was built using AppSheet integrated with Google Sheets.

Target Users: Teachers and staff of SMK Keningau II

  • Geofencing validation (200m radius)
  • Email domain validation (@moe-dl.edu.my)
  • Instant selfie capture via camera
  • Auto-block "Save" button outside geofence
  • Timestamped record with GPS coordinates

üõ† User Guide

  1. Log in using your official MOE-DL email.
  2. Fill in the sign-in form.
  3. Allow GPS location access.
  4. Capture a selfie using the camera.
  5. Press "Save" (enabled only if within the defined geofence).

Note: Users outside the permitted zone will not be able to save their attendance.
